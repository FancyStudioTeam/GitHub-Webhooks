# 🐙 GitHub Actions

A collection of custom GitHub Actions used in FancyStudio.

## ℹ️ Information

This GitHub Action executes a Discord webhook when a GitHub event is triggered. You must specify the events you want the GitHub Action to listen in the [`on`][GitHubWorkflowSyntaxOnPropertyUrl] property.

## 📋 Available Events:

- [ ] **Issues**:
    - [x] **Closed**
    - [x] **Opened**
